SLIDER CONTROL WITH THE HTML5 RANGE INPUT

A regular HTML5 silder

10% 150%

The code snipper for this slider is:

10% <input id="slide" type="range" min="10" max="150" step="5" value="100"/> 150%

The attributes within the slider element states:

  • type="range" - tells the browser to treat it as a slider input
  • the "min" and "max" values - sets the minimum and maximum range of the slider.
    Note that the '10%' and '150%' before and at the end of the code snippet has been added to 'see' the ranges of the slider.
  • step attribute - the 'jump' between the values along the range. So here it will be 10, 15, 20, 25 up to 150.
  • value attribute - the initial starting value.
